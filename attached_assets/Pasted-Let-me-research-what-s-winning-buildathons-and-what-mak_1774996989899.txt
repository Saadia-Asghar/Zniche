> ## STREAMING IMPLEMENTATION
>
> For all Claude API calls use Server-Sent Events (SSE):
> ```javascript
> // Backend route
> app.get('/api/build/stream/:step', async (req, res) => {
>   res.setHeader('Content-Type', 'text/event-stream');
>   res.setHeader('Cache-Control', 'no-cache');
>   res.setHeader('Connection', 'keep-alive');
>   const stream = await anthropic.messages.stream({...});
>   stream.on('text', (text) => res.write(`data: ${JSON.stringify({text})}\n\n`));
>   stream.on('end', () => res.write('data: [DONE]\n\n'));
> });
>
> // Frontend consumption
> const es = new EventSource('/api/build/stream/research');
> es.onmessage = (e) => {
>   if (e.data === '[DONE]') { es.close(); markStepComplete(); return; }
>   const {text} = JSON.parse(e.data);
>   outputPanel.textContent += text; // streams word by word
> };
> ```
